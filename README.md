Description: This project is an arithmetic expression calculator implemented in C++. It parses and evaluates expressions through tokenization, parsing, and evaluation.

Intended Use: The program is intended to be used for evaluating arithmetic expressions provided through a command line interface. It was created for educational purposes and used to demonstrate concepts like the software development cycle alongside tokenzing, parsing, and modular design.
